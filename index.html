Projects

WorkFlow (Freelance)

Since June 2023, I have been working on a personal project titled “Workflow - Desktop App”. This project, which demonstrates my technical skills in C#, .NET, Blazor, and Supabase, is aimed at updating an automotive body shop from paper record keeping to a digital format. The application, engineered using C# and Blazor, facilitates user input of customer and vehicle data in a collision repair shop environment. By utilizing the Supabase API for database management, the application ensures robust and secure storage of customer information along with user authentication. This project showcases my ability to develop efficient software solutions and manage databases effectively using modern APIs, while contributing to the digital transformation of traditional business processes.
